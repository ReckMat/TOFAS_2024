Conceitos de JavaScript

Saída com console.log()

O método console.log() é usado para exibir saídas no console do navegador.


// Exemplo
console.log("Programação");
let num = 1;
console.log(num);

Estruturas Condicionais (if/else)

As declarações if e if-else são usadas para tomar decisões baseadas em condições.


// Exemplo
let num = 5;
if (num > 10) {
    console.log("Maior que 10");
} else {
    console.log("Menor ou igual a 10");
}

Execução Sequencial

As instruções em um programa são executadas na ordem em que aparecem.


// Exemplo
let num = 1;
num = 2;
console.log(num); // Exibe 2

Operadores Aritméticos e Comparativos

Operadores aritméticos (+, -, *, /) e comparativos (==, >, <) são usados para realizar cálculos e comparações.


// Exemplo
let num1 = 10;
let num2 = 5;
console.log(num1 + num2); // 15
console.log(num1 > num2); // true

Declaração e Atribuição de Variáveis

Variáveis podem ser declaradas e valores podem ser atribuídos a elas.


// Exemplo
let num1 = 1;
let num2;
num2 = 2;
console.log(num1, num2); // 1, 2
num1 = 3;
console.log(num1); // 3
